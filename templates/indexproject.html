Welcome to AutoAware!

Our system, designed to excel in low-light object detection, is here to enhance truck driver safety by identifying obstacles using the robust YOLOv8 object detection model. This adaptive solution finds its place seamlessly across various industries:

  • Logistics: Streamline operations with enhanced safety measures.
  • Delivery: Safeguard shipments and drivers in all lighting conditions.
  • Construction: Navigate job sites securely, even in the dark.
  • Mining: Improve hazard recognition for miners' well-being.

Experience real-time protection on our website, where the magic of our technology comes alive. But that's not all – we offer an added advantage with the ability to upload videos for detection. By replicating real-world scenarios, we're unwavering in our commitment to enhancing safety across transportation-based industries. AutoAware is your ally for a safer journey.

YOLOv8

YOLOv8 is an advanced real-time object detection algorithm that stands for "You Only Look Once," version 8, enabling accurate and efficient recognition of objects in images and videos.

Flask

Flask is a lightweight and versatile Python web framework used for building web applications, APIs, and services, offering simplicity and flexibility for various development projects.

OpenCV

OpenCV (Open Source Computer Vision Library) is an open-source computer vision and machine learning software library. It provides a wide range of tools and algorithms for tasks like image and video processing, object detection, facial recognition, and more.

PyTorch

PyTorch is an open-source machine learning framework that provides a flexible and dynamic approach to building and training neural networks. It's widely used for tasks such as deep learning, natural language processing, computer vision, and other AI-related applications.
